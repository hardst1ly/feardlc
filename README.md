Fear DLC ⚡

Fear DLC — это Minecraft Java/Gradle клиентский проект, который делает упор на визуалы, модули, рендер, удобный интерфейс и стабильную работу.

О проекте ✨

Проект находится в разработке.
Главная цель — улучшать код, исправлять ошибки сборки, чистить структуру проекта и делать клиент стабильнее.

Визуалы 🎨

Все визуалы, стиль рендера, идеи интерфейса и визуальные модули сделаны от меня.

Сборка 🚀

Для сборки проекта:

./gradlew build

На Windows:

gradlew.bat build

Если тесты мешают сборке:

./gradlew build -x test
